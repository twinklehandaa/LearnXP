# LearnXP

> LearnXP is an interactive learning platform that allows users to track their learning progress, attempt quizzes, and earn rewards.

---

## Features

- **User Dashboard:** View name, referral code, total donations raised, and rewards.
- **Quizzes:** Attempt quizzes, track scores, and monitor progress.
- **Leaderboard:** Compare performance with other users.
- **Rewards System:** Earn badges and XP for completing activities.
- **Responsive Design:** Works on desktop and mobile devices.
- **Easy to Use:** Intuitive UI and navigation.

---

## Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Java (Servlets) 
- **Database:** Oracle
